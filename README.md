# Innovación Tech Consulting Website

This is the official website for Innovación Tech Consulting, a technology consulting firm specializing in transforming ideas into technological solutions that drive business growth.

## Features

This website includes the following sections:

*   **Hero Section:** A welcoming introduction to Innovación Tech Consulting.
*   **Value Proposition:** Highlights the company's strengths: proven experience, custom solutions, and cutting-edge technology.
*   **Services:** Details the services offered, including:
    *   Digital Transformation
    *   Big Data & Analytics
    *   Software Development
    *   Mobile Applications
    *   Strategic Consulting
*   **Blog Preview:** Showcases the latest blog posts on technology and innovation.
*   **Contact:** Provides a form and contact details (phone, email, location) to get in touch with the company.

## Technologies Used

*   **Next.js:** A React framework for building server-side rendered and static web applications.
*   **React:** A JavaScript library for building user interfaces.
*   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.
*   **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
*   **shadcn/ui:** A collection of re-usable UI components.
*   **Framer Motion:** For animations.

## Getting Started

To get a local copy up and running, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/innovacion-tech-consulting.git
    cd innovacion-tech-consulting
    ```
    *(Replace `your-username` with the actual username or organization if this is a public repository, otherwise, this instruction might need adjustment for a private context).*

2.  **Install dependencies:**
    This project uses `pnpm` as the package manager. If you don't have it installed, you can install it via npm:
    ```bash
    npm install -g pnpm
    ```
    Then, install the project dependencies:
    ```bash
    pnpm install
    ```

3.  **Run the development server:**
    ```bash
    pnpm dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
